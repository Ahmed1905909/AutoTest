Auto Test is a cutting-edge software application that helps testers to test their code before running their applications. With Auto Test, developers can easily and quickly identify and fix bugs, improve the quality of their code, and ensure that their applications are running smoothly. Auto Test's user-friendly interface makes it easy for developers of all skill levels to use. Simply upload your code and let Auto Test do the rest. The application will analyze your code, identify potential issues and provide detailed reports on where and how to fix them.



Our Features



Input


The user can choose the input type whether to upload java or python source code OR to write their own test scenario



Test Cases


AutoTest will generate reliable unit test cases from the user input



Testing


AutoTest will run the test cases on the user input and give the user the test results



Report


AutoTest will give the user a report with the test cases status after testing



History


The user can check previous tests through his History only if the user has an account on AutoTest
